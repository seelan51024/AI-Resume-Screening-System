Get it from: Google Account → Security → App Passwords
📖 How to set up Gmail SMTP
  1. Enable 2-Step Verification on your Google account
  2. Go to Google Account → Security → App Passwords
  3. Create a new App Password → select "Mail" → "Windows Computer" (or any)
  4. Copy the 16-character password
  5. Paste it in the App Password field above
  6. Click Save Config then Send Test Email to verify
